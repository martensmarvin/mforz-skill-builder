What does this skill do?

Describe the Zoho task you want Claude to learn. A skill is an instruction file that tells Claude when and how to execute a specific Zoho process.

This label is what you see in Claude. e.g. "zoho-lead-intake" or "crm-contact-enrichment"

Describe in 2–3 sentences what Claude should learn, and which Zoho actions it needs.

This helps Claude understand when the skill has been completed successfully

What should Claude report back? e.g. lead ID, company name, assigned rep.

💡 Why this matters
Een heldere skill-definitie is het verschil tussen een skill die Claude soms begrijpt en een skill die altijd werkt.

Vage skill-doelen → Claude raadt wat je bedoelt
Heldere skill-doelen → Claude weet precies wat te doen
📚 What you'll learn
Het definiëren van goede skills is de basis van effectieve Claude-automatisering. Deze skill-thinking kun je toepassen op elk Claude-project.

💡 Why skillsets and not individual actions?
Skillsets zijn vooraf samengestelde sets van acties die samen een logische workflow vormen. Ze geven je skill precies genoeg toegang — niet te breed, niet te smal. Verfijnen is altijd mogelijk via het uitklapbare paneel hierboven.
Step 4 of 6
Learning mode

What are the execution steps?

Skills contain logic. Describe step by step what Claude should do when this skill is triggered.

Use numbered steps with action verbs. Specify conditions: "If X, then Y."

Describe what Claude does if something is missing, duplicated, or fails. Use "If X, then Y."

📚 What you'll learn
Het afhandelen van uitzonderingen is wat een robuuste skill onderscheidt van een fragiele. Skills zonder uitzonderingslogica werken in zo'n 80% van situaties. Skills met goede uitzonderingsafhandeling werken in 95%+. Dit principe geldt voor elke Claude-skill die in productie draait.
Step 5 of 6
Learning mode

What guardrails apply?

Every skill has governance rules. Decide how autonomously this skill can operate and which actions always require confirmation.

New? Choose Sandbox. Only switch to Production after a successful test.

📚 What you'll learn
Guardrails are not one-size-fits-all. Different skills have different needs. Good skill designers ask themselves three questions: What is the risk? How often does this run? Are the actions reversible? This risk-thinking applies to every Claude skill with write access to Zoho.
